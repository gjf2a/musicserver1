Character Table for MusiSync Font (version 5.001)

You must have MusiSync installed to see the musical symbols. Without MusiSync, the cells will show only plain text.

Each cell contains a MusiSync symbol at size 36px. The symbols are padded by spaces in this table, so if you copy and paste, you will have unexpected extra space. Underneath, in red, is the alphanumeric character (from the Arial font). In blue is the decimal character code. Note that in the middle of the table, some of the decimal codes are not in numerical order: this is a Unicode thing.

Darkened cells have no symbol. These code locations cannot be used, for technical reasons.

Cells with pink background have symbols added in font version 5, but not present in earlier versions.

Some pink cells have a large asterisk (*). These are "uncommitted" locations. If you have a font editor and are of a mind to add your own symbols, use the uncommitted locations; but remember that the name of a revised font cannot contain "MusiSync" or anything confusingly similar.

Code 32 (character code number) is the ordinary space. Code 160 is the non-breaking space.

When you use MusiSync, you may find that consecutive lines overlap a bit. If so, you can either increase the line spacing, or insert dummy lines in an ordinary font.
